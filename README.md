### To clone this app
Type this into your terminal

    git clone https://github.com/adelicia-js/htmx-contact-app.git

### Run the app

#### Open app
    cd htmx-contact-app 
    
    code .

#### Set up venv 

    python3-m venv venv

    source venv/bin/activate

#### Install requirements

    pip install -r requirements.txt

#### Run app

    flask run

Navigate to url, tada :)
